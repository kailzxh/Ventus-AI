# AQI Prediction System

A real-time Air Quality Index prediction system using advanced machine learning models and React.js frontend.

## Features

- **Real-time AQI Monitoring**: Live data from multiple Indian cities
- **AI-powered Predictions**: NF-VAE model for accurate AQI forecasting
- **7-day Forecast**: Future AQI predictions with confidence intervals
- **City Comparison**: Compare air quality across different cities
- **Advanced Analytics**: Model performance and trend analysis
- **Responsive Design**: Mobile-friendly React.js interface

## Tech Stack

### Frontend
- React.js 18
- Tailwind CSS
- Recharts for data visualization
- React Leaflet for maps
- Axios for API calls

### Backend
- Python Flask
- PyTorch for NF-VAE model
- Scikit-learn for baseline models
- Pandas for data processing
- Kaggle datasets + Real-time API integration
